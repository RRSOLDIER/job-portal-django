# Job Portal Web Application (Django)

A full-stack Job Portal web application built using **Django**, supporting **role-based authentication** for Candidates and Recruiters.  
The platform allows recruiters to post jobs and manage applicants, while candidates can search, apply, and track application statuses.

This project follows **real-world software practices** and is designed as a **startup-style MVP**.
